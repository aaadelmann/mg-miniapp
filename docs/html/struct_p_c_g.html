Detailed Description

Preconditioned Conjugate Gradient (PCG) for SPD systems using MG V-cycles as right preconditioner.

Algorithm (Saad, Ch. 9):

\[ r_0=f-Au_0,\ z_0=M^{-1}r_0,\ p_0=z_0,\ \alpha_k=\frac{r_k^T z_k}{p_k^T A p_k},\ u_{k+1}=u_k+\alpha_k p_k,\ r_{k+1}=r_k-\alpha_k A p_k,\ z_{k+1}=M^{-1}r_{k+1},\ \beta_{k+1}=\frac{r_{k+1}^T z_{k+1}}{r_k^T z_k},\ p_{k+1}=z_{k+1}+\beta_{k+1}p_k. \]
