Detailed Description

Matrix-free geometric multigrid hierarchy with V-cycle preconditioner.

Operator
Anisotropic FD Laplacian applied by ::apply_A with no matrix assembly.
Workspaces
Per-level vectors (uL,fL,rL,tL) are scratch buffers used during preconditioning.
Parameters
nu1 / nu2 are pre/post smoothing steps; omega is the Jacobi weight.
