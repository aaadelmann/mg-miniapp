Detailed Description

Geometry-only description of one multigrid level.

Level \(\ell\) stores only sizes and spacings \((n_x^{(\ell)},n_y^{(\ell)},n_z^{(\ell)})\) and \((h_x^{(\ell)},h_y^{(\ell)},h_z^{(\ell)})\).
